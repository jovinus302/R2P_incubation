# Idea Candidates — Round B (테제 트랙) — 20260706-frontier-ai

> S2-T (Opus, 루브릭 v2 기준). 발산 제약: 테제 명시, wedge 의무, retention 구조, OpenClaw 비메신저 2개(T4/T5), PlaceLM 재설계(T1), PDE 미사용.

### T1: 플레이스 트레일 (PlaceLM 복원판) `[STRETCH]`
- **테제**: "추천은 죽었다. 낯선 남의 취향을 미는 대신, 내가 이미 발견하고 검증한 것을 상황에 맞게 복원하는 것이 맞다."
- **문제**: "장소는 계속 저장하는데, 정작 그 상황이 오면 뭘 저장했는지 하나도 기억이 안 난다."
- **타깃**: 지도 저장·방문이 수십~수백 개 쌓인 도시 생활자
- **컨셉**: "금요일 저녁, 강남, 2시간, 부모님 동행" → 내 저장/방문/사진 장소들로 동선·순서·이유(그때 왜 좋았는지)까지 복원. 남의 추천 0.
- **frontier 필수**: 저장 목록 + 방문 이력 + 사진 지오태그·캡션(비전)을 하나의 정합 서사로 엮는 추론은 이제서야 실용 수준.
- **근접·차별**: Ask Maps는 신규 추천형 — 본 제품은 "내가 검증한 것"의 상황별 복원.
- **wedge**: 지도 저장·타임라인 방문 이력 + 갤러리 위치태그 **일회성 임포트** → 첫 세션에 검증 장소 수백 개 즉시 확보.
- **retention 루프**: 상황(새 도시·주말·손님)마다 복귀 + 평소 사진·저장이 코퍼스 자동 증식 → 2주차에 더 정확.
- **삼성 자산**: 갤러리 위치태그, 온디바이스 지도 이력.

### T2: 곁눈 (실시간 다크패턴·사기 가디언) `[SAFE]`
- **테제**: "사기와 다크패턴은 인간이 눈치채기엔 너무 빨라졌다 — 결제·계약 화면은 이제 혼자 보면 안 된다."
- **문제**: "'무료체험' 눌렀다가 나도 모르게 매달 결제되고, 해지 버튼은 어디 숨었는지 못 찾겠다."
- **타깃**: 범용 일반인 (특히 부모님 세대)
- **컨셉**: 위험 화면(결제·약관·낯선 링크)을 실시간 공유하면 "이 무료체험은 3일 뒤 월 4만원 자동결제, 해지는 여기 숨어있음"을 짚어줌. 판단하는 눈, 차단기 아님.
- **frontier 필수**: 화면을 실시간으로 보며 "함정" 판단을 낮은 지연으로 — 멀티모달 GA 이후에만 가능.
- **근접·차별**: 애드블록/백신은 규칙 기반 — 처음 보는 화면의 의도를 실시간 해석.
- **wedge**: 라이브 화면 공유만 (cold-start YES).
- **retention 루프**: 결제·계약·낯선 링크 = 상시 위험 접점마다 자동 소환.
- **삼성 자산**: 온디바이스 화면 어웨어니스로 폰 기본 탑재.

### T3: 지치지 않는 대리인 (소비자 대리 투사) `[STRETCH]`
- **테제**: "기업은 당신이 지쳐서 포기하도록 환불·해지·보상을 설계했다 — 균형은 지치지 않는 대리인으로만 회복된다."
- **문제**: "환불받으려고 상담 3번 돌리고 서류 떼다 결국 포기했다."
- **타깃**: 범용 일반인
- **컨셉**: 이슈 붙여넣으면 브라우저 에이전트가 CS 플로우를 대신 항해, 근거 문구 작성·이의제기·에스컬레이션. 실행 전 제안-승인 게이트.
- **frontier 필수**: OSWorld 83.5%급 브라우저 조작 + 다단계 지속성.
- **근접·차별**: 환불 대행(사람 수동) 대비 24/7 자동 항해.
- **wedge**: 이슈 한 건 붙여넣기 (cold-start YES).
- **retention 루프**: 분쟁 반복 + 진행 케이스 추적·재촉 → 결과까지 매주 복귀.

### T4: 통근 집무실 (OpenClaw 자동차 게이트웨이) `[STRETCH]`
- **테제**: "운전하는 하루 1시간은 죽은 시간이 아니라 당신의 에이전트가 실세계를 실행하는 근무 시간이어야 한다."
- **문제**: "운전 중엔 손도 못 쓰는데, 처리할 일은 그때 다 떠오른다."
- **타깃**: 매일 왕복 통근 운전자
- **컨셉**: 차량에 격리된 로컬 OpenClaw Gateway — 음성 지시로 주행 중 스킬 실행(예약·비교·리서치·초안), 주차 시 요약·승인. 비메신저 임베드.
- **frontier 필수**: 30분 창 안 다단계 자율 실행 지평이 최근 성립.
- **근접·차별**: 차량 음성비서는 내비·전화 수준 — 실제 웹 태스크 완수.
- **wedge**: 세션 음성 태스크 + 캘린더·내비 선존재 맥락 (PARTIAL). 격리·큐레이션 스킬스토어가 해자 (ClawHavoc 대응).
- **retention 루프**: 하루 2회 통근 고정 슬롯 — 출근길 지시, 퇴근길 승인.
- **삼성 자산**: Harman 오토모티브.

### T5: 거실 에이전트 (OpenClaw TV 가족 공용) `[STRETCH]`
- **테제**: "가족의 디지털 실행은 각자 폰 속에 숨지 말고 거실로 나와 감독받아야 한다 — 앱스토어의 시대는 끝나고 스킬이 하드웨어로 들어온다."
- **문제**: "여행·경조사·집안 관리 예약을 누가 뭘 했는지 가족끼리 매번 엇갈린다."
- **타깃**: 공동 로지스틱스 많은 가구
- **컨셉**: TV에 격리 로컬 Gateway — 가족 누구나 음성 태스크, TV 화면에 공용 태스크 큐 상시 노출, 승인·추가. 비메신저 임베드.
- **frontier 필수**: 로컬 에이전트 런타임 + 스킬 마켓 조합이 1년 미만. SwitchBot 선례로 임베드 경로 검증.
- **근접·차별**: SwitchBot은 홈 자동화 중심 — 본안은 가족 공용 실세계 태스크 실행+감독 서피스.
- **wedge**: 큐레이션 스킬 프리로드 + 세션 지시 (PARTIAL). 화이트리스트·격리 실행이 상용화 전제 = 격리가 곧 컨셉.
- **retention 루프**: 거실 지나며 매일 공용 큐 확인·승인·추가.
- **삼성 자산**: TV 플릿 + 가전.

### T6: 삶의 증거 검색 (갤러리 심문기) `[SAFE]`
- **테제**: "당신의 사진 5만 장은 넘겨보는 추억이 아니라, 아직 아무도 심문하지 않은 당신 삶의 증언이다 — 기억은 검색되어야 한다."
- **문제**: "우리 애 언제 처음 걸었지, 이 증상 언제부터였지 — 사진에 다 있는데 못 찾는다."
- **타깃**: 범용 일반인 (사진 수천~수만 장)
- **컨셉**: 갤러리에 묻면 종단 추론으로 답 — "작년 대비 외식 2배", "이 화분 6주째 시듦", "걸음마 첫 프레임은 이날". 검색이 아니라 삶의 질문.
- **frontier 필수**: 수만 장 비전 + 장기 컨텍스트 종단 추론이 이제야 실용.
- **근접·차별**: Google Photos Ask는 단건 검색 — 시간축 추세·변화 서술.
- **wedge**: 갤러리 (대량 선존재) **일회성 인덱싱** → 첫 질문 즉시 wow (YES).
- **retention 루프**: "언제부터/얼마나 자주" 질문 반복 + 사진 매일 축적.
- **삼성 자산**: 갤러리, 온디바이스 비전.

### T7: 자는 동안 (야간 자율 에이전트) `[WILD]`
- **테제**: "가장 비싼 자원은 시간이 아니라 '지켜봐야 함'이다 — 이제 위임하고 자면, 아침에 결과가 와 있어야 한다."
- **문제**: "리서치·비교·신청 같은 몇 시간짜리 일을, 붙어서 지켜볼 시간이 없어 계속 미룬다."
- **타깃**: 범용 일반인
- **컨셉**: 밤에 태스크 큐 채우고 자면 에이전트가 밤새 다단계 실행, 아침에 "밤새 한 일 + 승인 필요" 브리핑.
- **frontier 필수**: 5시간대 자율 지평(METR, 배가 3–4개월)이 말 그대로 최근에 열림.
- **근접·차별**: Manus류는 세션 감독 전제 — 취침이라는 비감독 배치 슬롯 특화.
- **wedge**: 세션 태스크 큐 입력 (YES). 신뢰도 리스크 → 승인 게이트 필수.
- **retention 루프**: 잠은 매일 — 밤마다 큐 채우는 습관이 구조적.

### T8: 진짜 확인 (실시간 진위 가디언) `[WILD]`
- **테제**: "AI가 만든 시대엔 무엇도 진짜라 믿을 수 없다 — 의심스러운 목소리·영상 앞에 실시간 '이거 합성이야'를 말해주는 존재가 기본값이어야 한다."
- **문제**: "부모님 목소리로 급하게 돈 보내라는 전화 — 진짜인지 그 순간엔 알 방법이 없다."
- **타깃**: 범용 일반인
- **컨셉**: 통화·영상·미디어를 실시간 판정 — 합성 흔적 + 맥락 판단(모르는 번호+사칭 패턴+아티팩트) 결합 경고.
- **frontier 필수**: 실시간 멀티모달 합성 탐지 + 맥락 추론 결합, 온디바이스.
- **근접·차별**: 기존 탐지기는 오프라인 단건 — 통화 중 실시간 개입.
- **wedge**: 라이브 통화/미디어만 (YES). 정확도 군비경쟁 → WILD.
- **retention 루프**: 의심 통화·영상 상시 유입 = 매 위험 접점 소환.
- **삼성 자산**: 통화·온디바이스 처리 폰 기본 탑재.
