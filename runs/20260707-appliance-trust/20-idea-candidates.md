### Idea 1: TrustGate (검증 행동 가드)  `[STRETCH]` `[D]`
- **문제**: "에이전트가 우리 집 오븐을 밤새 켜두라 했는데, 그게 진짜 안전한 판단인지 내가 어떻게 믿어?"

- **제품 컨셉**: 에이전트가 가전에 물리 명령을 내리는 순간, 가전 실리콘이 "지금 실제 상태 + 정책"에 비춰 명령을 검증하고, 환각·탈취·위험 명령을 실행 직전에 차단하며 사람이 읽는 근거를 보여줌("오븐이 비었고 3시간 무명령 → 화재 위험, 차단"). 사용자는 안심 토글 하나와 차단 로그만 봄.
- **why this tech now**: HomeBench에서 GPT-4o 다중기기 무효 명령 성공률 0%, ClawJacked(CVE-2026-32025)로 클릭 없이 게이트웨이 탈취 실증 — 에이전트가 물리 세계를 만지기 시작한 지금에야 "행동 검증 레이어"가 필수.
- **근접 제품 + 차별점**: Gemini for Home/Alexa+ — 제어는 하되 검증 계층 없음. 명령을 믿고 실행이 아니라 실물 상태로 반증.
- **moat 가설**: 실리콘 신뢰 루트에 정책 엔진을 붙인 통합 깊이 — 앱 계층 경쟁자는 상태를 위조·환각 없이 검증할 구조가 없음. 표준화 시 서드파티 스킬 전환비용.
- **경쟁 킬 테스트**: 구글이 "안전 확인" 문구를 붙여도 attested 상태 없는 소프트 LLM 체크(스스로도 환각) → 삼성엔 하드웨어 반증이 남음.
- **cold-start 데모**: YES — 실시간 상태만 필요.

### Idea 2: SafeProof (홈 신뢰 증명서 — 보험)  `[STRETCH]` `[S]`
- **문제**: "우리 집은 실제로 안전하게 관리되는데 보험사엔 증명할 방법이 없어 할인도 못 받고, 사고 나면 입증 싸움을 해."
- **타깃 유저**: 주택보험 가입 자가 소유주(누수·화재 민감) + 손해율 낮추려는 손보사.
- **제품 컨셉**: 가전·센서가 지금의 안전 상태(누수감지기 정상·건조, 인덕션 OFF, 도어락 잠금)를 실리콘 서명된 검증가능 크리덴셜로 발급 → 보험사에 제출해 즉시 할인 자격/간편 청구. FNOL 시 "사고 시점 실제 상태" 서명 증거 자동 첨부.
- **why this tech now**: attested 실시간 상태 서명 가능(Knox Vault 가전 확장) + Hippo Clara류 AI FNOL이 검증 입력 필요 + 누수감지 5–12% 할인 관행 실재.
- **근접 제품 + 차별점**: 기존 스마트홈 할인은 자기신고/앱 로그 — 위조 불가능한 실리콘 서명이라 보험사가 신뢰.
- **moat 가설**: (복리) 서명 상태를 인수·정산에 채택하는 보험사 네트워크 + 커버리지 확대 → 후발 생태계는 처음부터 재설득.
- **경쟁 킬 테스트**: 구글/애플은 가전을 제조·서명 못 함 → 서명 발급 주체 불가. 보험 앱이 흉내내도 신뢰 근거 없음.
- **트랙 S 근거**: 핵심 베팅 = "손보사가 attested 상태를 인수·정산 근거로 채택" — 6개월 내 보험사 1곳 파일럿/할인 적용 신호로 판정.
- **cold-start 데모**: PARTIAL — 첫 wow는 "지금 상태 스냅샷 서명 → 할인 자격 즉시 표시"(wedge). 정산 신뢰는 복리 메커니즘.

- **문제**: "우리 집에 이런저런 AI 스킬을 깔았는데 뭘 읽고 뭘 만졌는지·나쁜 짓 안 했는지 확인할 방법이 없어."
- **타깃 유저**: 커뮤니티 에이전트 스킬(OpenClaw류) 실험 파워유저 + 프라이버시 민감 가정.
- **제품 컨셉**: 각 에이전트/스킬이 집 상태에 무엇을 읽고 어떤 물리 행동을 했는지 실리콘 서명된 위조불가 원장으로 기록·표시, 권한을 암호학적으로 즉시 취소. ClawHavoc식 악성 스킬 방화벽.
- **why this tech now**: ClawHavoc(악성 스킬 341개/11.9%)·ClawJacked 실증 + 통제권 없음 우려 47% — hair-on-fire.

- **moat 가설**: 실리콘 강제 권한 + 서명 원장 통합 — 앱 계층 권한 UI는 스킬이 우회·위조 가능.
- **경쟁 킬 테스트**: 구글 권한 대시보드는 소프트 권한이라 우회 → 실리콘 강제만 남음.
- **cold-start 데모**: YES. 원장 가치는 세션 내 축적(wedge).

### Idea 11: 홈증인 (분쟁 증거)  `[WILD]` `[D]`
