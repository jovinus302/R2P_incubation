# Idea Card 형식 (S2 산출 단위, 카드당 ~10줄)

### Idea <n>: <작업명>  `[SAFE | STRETCH | WILD]`
- **문제** (1문장, 사용자 입으로):
- **타깃 유저** (구체적 페르소나, "모두" 금지):
- **제품 컨셉** (2–3문장 — 보고 만지는 것, 아키텍처 아님):
- **why this tech now** (1–2문장):
- **근접 제품 + 1줄 차별점**:
- **moat 가설** (1줄, 유통·기본 탑재 제외 — 복리 데이터 루프/통합 깊이/전환 비용 중 무엇):
- **경쟁 킬 테스트** (1줄 — 구글/애플이 6개월 내 뻔한 버전 출시 → 남는 것):
- **트랙 제안**: D(데모) / S(전략 베팅 — 왜 D로 못 하는지 1줄 필수)
- **cold-start 데모**: YES / PARTIAL / NO (첫 wow까지 필요한 사용자 데이터 1줄, NO면 wedge 후보 1줄)
